# Register layout XML files

## How can layout XML files be registered for the frontend and adminhtml areas?

```xml
<frontend>
  <layout>
      <updates>
          <catalog>
              <file>catalog.xml</file>
          </catalog>
      </updates>
  </layout>
```
или
```xml
<adminhtml>
  <layout>
      <updates>
          <catalog>
              <file>catalog.xml</file>
          </catalog>
      </updates>
  </layout>
```
